Debugging and Pseudo Coding Assignment

Correct the errors in the pseudocode and create flowcharts for each application.

1. Sale Discount Program

Original Pseudocode:

// This pseudocode is intended to describe
// computing the price of an item on sale for 10% off

start
   input origPrice
   discount = price * 0.25
   finalPrice = origPrice - discnt
   output finalPrice
stop

Corrected Pseudocode:

start
   input origPrice
   discount = origPrice * 0.10
   finalPrice = origPrice - discount
   output finalPrice
stop

2. Miles Per Gallon Program

Original Pseudocode:

// This pseudocode is intended to compute the number
// of miles per gallon you get with your automobile.

start
   input milesTraveled
   input gallonsOfGasUsed
   milesPerGallon = milesTraveled + gallonsOfGas
   output milsperGallon
start

Corrected Pseudocode:

start
   input milesTraveled
   input gallonsOfGasUsed
   milesPerGallon = milesTraveled / gallonsOfGasUsed
   output milesPerGallon
stop

3. Rent Cost Per Day Program

Original Pseudocode:

// This pseudocode is intended to describe
// computing the per day cost of your rent
// in a 30-day month

start
   input rent
   costPerDay = rent / 31
   output rent
stop

Corrected Pseudocode:

start
   input rent
   costPerDay = rent / 30
   output costPerDay
stop

4. Discretionary Spending Program

Original Pseudocode:

// This program accepts a user's monthly pay
// and rent, utilities, and grocery bills
// and displays the amount available for discretionary spending

start
   input pay
   input rent
   input utilities
   input groceries
   discretionary = pay - rent - utlilites - groceries
   output discretionary
stop

Corrected Pseudocode:

start
   input pay
   input rent
   input utilities
   input groceries

   totalBills = rent + utilities + groceries
   discretionary = pay - totalBills

   output pay
   output totalBills
   output discretionary
stop
